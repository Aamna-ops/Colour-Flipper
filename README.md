# Colour Flipper

A simple interactive web project built using HTML, CSS, and JavaScript that allows users to change the background colour of the page using buttons or generate a random colour.

## Project Overview
The project includes buttons for preset colours (red, green, and blue) as well as a random colour option. When a button is clicked, JavaScript dynamically updates the page background colour.

## Technologies Used
- HTML
- CSS
- JavaScript

## Key Features
- Button-controlled colour changes using JavaScript functions
- Random colour generation using RGB values
- Dynamic DOM manipulation
- Simple, responsive user interface

## What I Learned
- How to use JavaScript variables and functions
- Manipulating the DOM to change page styling
- Using `Math.random()` to generate dynamic values
- Connecting JavaScript logic to HTML events

## Possible Improvements
- Display the RGB colour value on screen
